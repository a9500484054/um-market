Домен — это адрес сайта, по которому сайт можно найти в сети. Если вы хотите подключить собственный домен (то есть чтобы сайт открывался по адресу вида http://mysite.com), нужно сделать 3 шага:

  • 1

    Купить домен

    Купить домен можно у регистраторов доменов. По запросу «купить домен» можно легко найти их в поисковиках в интренете.

  • 2

    Указать домен в настройках сайта

    Внесите свой домен в поле «Подключить свой домен» и нажмите сохранить. Система подскажет вам оптимальный для вашего сайта IP-адрес А-записей, которые нужно установить на стороне регистратора

  • 3

    На сайте регистратора указать записи для домена

    Для этого в зоне управления DNS нужно добавить запись A и прописать для нее значение IP адреса из Настроек сайта → Домен
